# SEO Optimization Tool

## 📋 프로젝트 개요
- 네이버 스마트스토어 상품 SEO 최적화 도구
- 개인 상품 관리 및 SEO 점수 분석

## 🎯 주요 기능
- [ ] 상품 SEO 점수 분석
- [ ] 대시보드 시각화
- [ ] 상품 정보 관리

## 🛠 기술 스택
- Backend: Python (FastAPI)
- Frontend: React
- Mobile: Flutter
- Database: PostgreSQL (Supabase)
